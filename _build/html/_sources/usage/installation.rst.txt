Installation
============

Downloading Chamelon
-----------------

The source code of Chamelon can be found at `<https://github.com/Ekdohibs/chamelon>`_.
Use Git to get access to the latest version:

.. code-block:: shell-session

   $ git clone https://github.com/Ekdohibs/chamelon.git

Installing Chamelon from source
----------------------------

 In an Opam switch with compiler OCaml 4.14.0, run:

.. code-block:: shell-session

   $ dune build
